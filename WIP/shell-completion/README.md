# Shell completion for Clamp — working notes

This directory tracks the incremental development of shell completion
generation for Clamp (issue #45).

## Process

We're working in TDD increments. Each increment:

1. Discuss next steps
2. Write tests
3. Write code to get tests passing
4. Consider refactoring
5. Append notes to `progress.md`
6. Commit to the feature branch

## Files

- `plan.md` — overall design and approach
- `progress.md` — per-increment notes
